# 必須繳交知名魔法書才能被允許進入
1. 高大的城牆
2. 魔法屏障
3. 入口有守衛把手
# 職位
1. 守書人，宣揚者
2. 大閱讀者協會，類似議會，有八名成員
3. 首席閱讀者，協會主席
# 規則
1. 不能動武，紛爭只能用辯論解決
2. 不能偷竊
3. 不能擅自複製書籍，得付費
4. 不能損壞或修改作品
5. 比燭火更旺的火焰都會被結界阻擋
# 設施

	1. 空蕩蕩
2. 澡堂
3. 旅店
4. 抄寫室
5. 圖書館
6. 研究室
