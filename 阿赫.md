# 男性豺狼人
雖然是身材高大的豺狼人，有著鋒利的爪子，但是他有一雙和善的深褐色眼眸。土黃色毛皮，深色斑點，深色吻部，土灰色鬃毛，尾巴時不時會自己搖擺起來。

# 信仰伊爾瑪特的牧師
頭戴荊棘頭冠，一看就非常不舒服，雙手手臂纏繞象徵受難之神伊爾瑪特的紅線，身穿藍色僧侶袍，外面罩著一件鎖子甲，袍子底下露出一雙赤裸的腳掌，一副苦行僧模樣。身後背負一根由數枝荊條擰成一束的狼牙棒，上面的尖刺因鮮血而染紅；左手持握一面布滿刀痕、明顯經歷風霜的盾牌

# 講述自己的故事
1. 我的部族屠殺了一個村莊，我也有參與其中。當我踏進村子，眼前都是殘破的軀體，耳邊盡是遮掩不住的哀嚎。我嚇得無法呼吸，每根毛都在顫抖。族長看我這麼軟弱的樣子，就把我帶到一名提夫林小女孩面前，要我把他殺了。可是我就是沒辦法下手，連刀子都拿不穩，那時候我不知好死，竟然脫口說出搶走糧食和金幣就好，不需要殺人的。在豺狼人的世界裡，是不允許弱者活下去的，所以我就被族人打個半死，丟在路邊等死。幸好索菲亞救了我，索菲亞是我的恩人，在我快死掉的時候撿回來，幫我療傷。他的家人在那場屠殺中全數喪生，無一倖存。我猜想索菲亞每次看到我，都會想起他家人的慘樣。他一定很恨我，可還是收留了無家可歸的我。我很納悶，過了很久我問他為什麼那時要救我，我印象很深刻，他對我說「因為你也在受苦啊，如果我無視你的痛苦，誰來幫你呢？我的孫女生前一定希望有人能幫他，而你有試著去幫他，對我來說就值得感到欣慰了。」

# 當有人尋求安慰時
1. 你所遭受的苦難，並非毫無意義，回顧你的一生，便會發現，就是這些苦難使你成長茁壯。你所受到的不公平對待，讓你理解公平正義的價值；你所遭到的背叛，讓你學會忠誠的重要；你所感到的孤獨，讓你不會將朋友視為理所當然；而你所遭逢的厄運，使你能夠體會機運也是人生的一部分；當你受到別人的冷漠對待時，你會發覺關懷的重要性；而別人對你的忽視，使你能夠學會傾聽他人的聲音；你所碰上的苦難，讓你知道何謂同情。就算不希望壞事降臨，壞事也會找上門來，至於能不能從這些壞事中學到東西，就是我們解開不幸背後的人生課題
2. 你遇到的磨難都是上天給予你的考驗。記得以前你小時無知，被火焰迷住嘗試碰觸的時候，發生了什麼事嗎？你痛得縮回了手，從此你學到了如何避免危險

   他曾在苦痛中呻吟，在黑暗中受難，卻未曾遺失人心的微光，願您伸出那滿布傷痕的手，撫平他破碎的靈魂。
   主啊，讓他在您無聲的恩慈中安息，他的痛苦已盡，折磨已止，如今只剩平靜與光。
   他的勇氣與仁慈，如聖痕銘刻在我們心中，提醒我們：痛苦雖深，憐憫更深；折磨雖久，慈悲無盡。
   主啊，請看顧我們的淚水，也教導我們，如您那般為他人承擔，為痛苦者守望。
   願伊爾瑪特垂憐，願亡者安寧。
