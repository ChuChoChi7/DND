# [[碎骨盟]]四天王之一
# 渡鴉人斥侯
# 調查出骨王座[[亞伯拉罕]]實為[[神聖聯合大公國|神國]]人
1. 可能從骨王座[[亞伯拉罕]]的貴族宅邸脫逃出來，然後為了躲避腐爛鷹人和狗活屍，逃進下水道。
	1. 來源：[[阿魯]]偵測思想
2. 甦醒後告知[[亞伯拉罕]]實為[[神聖聯合大公國|神國]]人
# 被救出來回到[[舊綠皮堡]]
1. 一行人在下水道發現昏迷的他，無論怎麼嘗試都無法清醒過來，只得帶回[[碎骨盟]]治療
# 靈魂補齊
1. 被[[綠皮聯盟]]薩滿[[草寇]]診斷出，因為靈魂不完整才昏迷不醒
2. 在一行人幫助下湊齊靈魂碎片，終於清醒。
