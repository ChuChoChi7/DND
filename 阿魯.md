# 背景故事
阿魯的家鄉──莫查查洞窟，分為上下游兩大區域，由一條地下河川貫穿。上游由狗頭人佔據，下游則是蕈菇人部落的棲身之地。長年以來，狗頭人為了爭奪食物、資源，不斷襲擊蕈菇人，將他們擄走。阿魯也不幸被俘，關進牢籠，等待著成為食物的命運。

正當他絕望之際，兩頭兇猛的食腐獸闖入洞窟，一口吞下了一名狗頭人。狗頭人奮起反抗，歷經過半傷亡才終於擊殺其中一隻，然而另一頭食腐獸仍虎視眈眈、毫髮無傷。那隻死去的食腐獸倒在阿魯面前，而另一隻正揮舞著觸手朝牢籠逼近。情急之下，阿魯撒下寄生胞子，附著在屍體上，將其操控為傀儡猛然撲出。

突如其來的「詐屍」嚇得食腐獸驚駭倒退，也為狗頭人爭取到反擊的契機。倖存的狗頭人一擁而上，將最後的敵人亂棍打死。戰後，狗頭人們面面相覷，不知該如何面對眼前這位翻盤關鍵的蕈菇人。這時，狗頭人族長霍克站了出來，打開牢門，釋放所有蕈菇人，並向族人宣布：

「這些是我們的救命恩人，從今以後，我們要善待他們，與他們和平共處。」

說罷，他讓阿魯一行人自由返回。

阿魯凱旋歸來，受到了英雄般的熱烈歡迎，甚至被推舉為新任族長。

歲月流轉，莫查查洞窟中的蕈菇人與狗頭人和平共處多年，偶有交流，生活安定。然而某日，不知何故，狗頭人突然對蕈菇人發動猛烈突襲。雙方爆發激烈衝突，雖成功擊退敵人，但蕈菇人傷亡慘重。

戰後不久，部落便遭遇糧食歉收，族人接連染病。經調查，發現地下河遭到污染。阿魯立即組織小隊，循河而上，前往源頭調查污染源並尋求解決之道。

他們途經狗頭人領地時，卻發現整個區域滿目瘡痍，屍橫遍野，找不到任何生還者，連族長霍克也下落不明。

小隊一路艱辛跋涉，最終來到地表世界。他們震驚地發現，附近已被開發成一座神國城市。大量廢棄物被棄置於森林中，惡臭的廢水則直接排入河川，正是污染的根源。

就在此時，小隊被巡邏的城市警備隊發現。突如其來的襲擊中，大多數成員不幸慘死。阿魯帶領少數倖存者逃離，但途中與夥伴失散，最終孤身一人，跌跌撞撞逃入了未知的魔物之巢……
