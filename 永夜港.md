# 位置
1. [[魔物之巢]]最西邊海邊
2. 距離[[麥爾斯鎮]]六天路程
3. 距離[[舊綠皮堡]]七天路程

# 不夜城
1. 用強大的魔法結界隔絕日光
# 四天王
1. 領主血王座[[雷威克]]，掌管著港內血奴的治權與北側的貴族區
2. 骨王座[[亞伯拉罕]]，統領西側與南側，掌管毒品與民生物資。實為[[神聖聯合大公國|神國]]人
3. 鐵王座[[安士姆]]，統領東側，掌管兵器與軍權
4. 棘王座[[艾蓮娜]]，統領中央地帶的花街，也就是紅燈區。非血族
5. 傳聞有第五王座，被趕去下水道
# 禁止爭鬥
1. 任何一點爭鬥都會被抓走關進鐵王座大牢
# 血族、血奴
1. 四天王皆為血族
2. 血奴可以是任何種族，提供血液和勞動
# 花街
1. 禁止使用魔法，魔法一施展立刻消散
2. 所有種族，包含[[神聖聯合大公國|神國]]人都能來玩
3. 三大花魁
	1. 卓爾女性[[黑蓮花]]
	2. 狐人女性[[雪百合]]
	3. 火元素裔女性[[小菊花]]
# 下水道
1. 錯綜複雜
2. 充滿垃圾、難聞的氣味
3. 流浪漢聚集地
